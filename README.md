LimbusdlgBubble_RU-ATRu-or-RTRu-
Перевод с английского на русский с использованием ненормативной лексики.

Способ установки:
Скачайте BattleSpeechBubbleDlg.json и переместите его содержимое в папку Lang (в любой локализатор). Если вы хотите играть с оригинальным встроенным переводом (например, английским), просто создайте папку в Lang, и назовите папку как угодно, и вставьте туда BattleSpeechBubbleDlg.json. Выберите при запуске игры папку с вашим названием
Папка Lang находится по пути: Limbus Company -> LimbusCompany_Data -> Lang. Чтобы найти папку с игрой, перейдите в библиотеку Steam, откройте страницу игры, нажмите на значок шестерёнки, выберите «Управление» — «Просмотреть локальные файлы».
