LimbusdlgBubble_RU-ATRu-or-RTRu-
Перевод с английского на русский с использованием ненормативной лексики.

Способ установки:
Скачайте ZIP-архив и переместите его содержимое в папку Lang (в любой локализатор). Если вы хотите играть с оригинальным встроенным переводом (например, английским), просто скопируйте папку в Lang и выберите при запуске игры папку с dlg.
Папка Lang находится по пути: Limbus Company -> LimbusCompany_Data -> Lang. Чтобы найти папку с игрой, перейдите в библиотеку Steam, откройте страницу игры, нажмите на значок шестерёнки, выберите «Управление» — «Просмотреть локальные файлы».
